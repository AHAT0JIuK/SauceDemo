# PageObject

0. **BaseTest** — инициализация тестового браузера и страниц для проверок.

1. **LoginTest** — покрытие проверками страницы залогина:

    * ввод валидных и невалидных данных.

2. **ProductsPageTest** — покрытие проверками страницы с продуктами:

    * открытие страницы;
    * открытие сайдбара;
    * кнопка добавления продукта в корзину.

3. **ItemPageTest** — покрытие проверками страницы продукта:

    * открытие страницы;
    * возврат на страницу с продуктами;
    * добавление продукта в корзину.

4. **CartPageTest** — покрытие проверками страницы продуктовой тележки:

    * открытие страницы;
    * возврат на страницу с продуктами;
    * переход на страницу Checkout.

5. **CheckoutPageTest** — покрытие проверками страницы Checkout:

    * ввод валидных и невалидных данных.

6. **CheckoutOverviewPageTest** — покрытие проверками страницы CheckoutOverview:

    * открытие страницы;
    * переход на страницу с продуктами;
    * переход на страницу тележки.

7. **FinishPageTest** — покрытие проверками последней страницы оформления заказа:

    * открытие страницы;
    * переход на страницу с продуктами;
    * переход на страницу тележки.
